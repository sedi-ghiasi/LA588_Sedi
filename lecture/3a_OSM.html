Adding an OpenStreetMap to an HTML page

Lecture 3a

In this scenario, I wanted to share the location of a fictional area of concern.
Maybe this is a damaged sidewalk or a bad pothole

One method to do this would be to just share the URL link to an OSM map.
https://www.openstreetmap.org/?mlat=42.51115&mlon=-94.17502#map=18/42.51116/-94.17502&layers=N

Note an alternative to that URL woudl include a parameter of CN for the layers as below. When we pass variables to a mapping application we can do this in the URL using a GET() statement or as values submitted from a form using a POST() statment. There are other methods, but these are the two primary protocols.
Take note that in this second link I utilized a Target parameter for the hyperlink so that the browser would open my map in a new window.

https://www.openstreetmap.org/?mlat=42.51115&mlon=-94.17502#map=18/42.51116/-94.17502&layers=N

Or you could embed the map using what is called an iframe
